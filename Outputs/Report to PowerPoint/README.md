## Export a Report to PowerPoint
### Grab screenshots (PNG) per Report tab / page and create a PPTX file with 1 tab/image per slide.

### Use it in combination with the Project metadata block to obtain the desired Report tabs URLs

### Requires the installation of Chromedriver on your machine. Download it from [here](https://sites.google.com/a/chromium.org/chromedriver/downloads).
